# Roadmap (indicative)

- Optional **on-device model** adapter interface (no weights in-repo).
- Richer **agent sandbox** hooks (storage quotas, optional network gates).
- **Desktop / JVM** flavor of agent-sdk for non-Android hosts.
- Sample **instrumented tests** for `AgentRuntime` and mocks. *(JVM unit tests for `DefaultAgentRuntime`, mocks, and example agents ship in CI.)*
- Hardened **proguard/R8** configs for release builds.

Priorities may change; this file is not a commitment of service.
