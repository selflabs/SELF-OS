# Overview

**SELF Shell** is an open-source personal intelligence shell for Android. It provides:

- A **Compose-based shell UI** (dashboard, agents, apps, settings).
- A local **agent runtime** and **plugin registry** for SELF Apps.
- **Mock** implementations of Harmony Mesh, wallet, app store, and Aura-style consent previews.
- **Example agents and apps** with no network dependencies.

It is **not** the full production SELF OS experience. See [OPEN_SOURCE_BOUNDARY.md](OPEN_SOURCE_BOUNDARY.md).

**Tagline:** Build your own personal AI system — or connect to the full SELF OS ecosystem when you are ready.
